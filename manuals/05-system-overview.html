System Overview

Learn about the main menu structure and navigation in Mana MES

เมื่อเข้าสู่ระบบ Mana MES ผู้ใช้งานจะพบกับเมนูหลักทางด้านซ้ายมือ คู่มือนี้จะช่วยให้คุณเข้าใจโครงสร้างเมนูและวิธีการนำทางในระบบ

แถบควบคุมเมนูหลัก

เมื่อเข้าสู่ระบบ ผู้ใช้งานจะพบกับเมนูหลักทางด้านซ้ายมือ ซึ่งแบ่งออกเป็นหมวดหมู่ดังนี้:

📋 หมวดหมู่เมนูหลัก
•
สถานะการทำงาน (WIP and OEE)

ดูภาพรวมการผลิตแบบ Real-time

•
ข้อมูลการผลิต (Production Data)

จัดการใบสั่งงาน (WO) และข้อมูลสินค้า (SKU)

•
ข้อมูลอุปกรณ์ (Equipment Data)

จัดการข้อมูลเครื่องจักรและการหยุดเครื่อง (Downtime)

•
บันทึกกิจกรรม (Activity Log)

ดูประวัติการใช้งานระบบ

System Overview - แถบควบคุมเมนูหลัก

การเปลี่ยนภาษา

เข้าสู่เมนูเปลี่ยนภาษาด้วยเมนูโปรไฟล์ มุมบนขวา

📍 เส้นทางการนำทาง

เมนูโปรไฟล์ (มุมบนขวา) → Language Settings

1

ค้นหาไอคอนเมนูโปรไฟล์ที่มุมบนขวาของหน้าจอ

2

คลิกที่เมนูโปรไฟล์เพื่อเปิดเมนูแบบเลื่อนลง

3

เลือกภาษาที่ต้องการจากตัวเลือกภาษา

System Overview - การเปลี่ยนภาษา

ข้อมูลอ้างอิงด่วน

เคล็ดลับการนำทางเมนู

  • • เมนูหลักจะแสดงอยู่ทางด้านซ้ายของหน้าจอเสมอ เพื่อให้เข้าถึงฟีเจอร์ทั้งหมดได้ง่าย
  • • แต่ละหมวดหมู่เมนูจะจัดกลุ่มฟังก์ชันที่เกี่ยวข้องกันไว้ด้วยกันเพื่อการจัดระเบียบที่ดีขึ้น
  • • ใช้ Activity Log เพื่อตรวจสอบประวัติการใช้งานระบบและติดตามการเปลี่ยนแปลง
  • • สามารถเปลี่ยนภาษาได้ตลอดเวลาผ่านเมนูโปรไฟล์โดยไม่กระทบต่อข้อมูลของคุณ
