Monitoring Production Performance

Track OEE metrics and manage downtime

Mana MES มีเครื่องมือติดตามที่ครอบคลุมเพื่อวัดผลประสิทธิภาพโรงงานตามมาตรฐานสากล และการดูภาพรวมการผลิตของโรงงาน

คู่มือนี้ครอบคลุมวิธีการอ่าน OEE Dashboard metrics และวิธีการเพิ่มบันทึก Downtime ด้วยตนเอง

OEE Dashboard (แดชบอร์ดข้อมูลเชิงลึก)

หน้าจอสรุปผลสำหรับผู้จัดการ เพื่อดู "สุขภาพ" ของโรงงาน

📍 เส้นทางการนำทาง

ค่าที่แสดงผล

แดชบอร์ดแสดงค่าต่อไปนี้:

🎯

OEE (Overall Equipment Effectiveness)

ประสิทธิผลโดยรวมของเครื่องจักร

เป้าหมายมาตรฐาน: 85%

นี่คือตัวชี้วัดหลักของประสิทธิภาพการผลิตของโรงงาน

⏰

Availability (ความพร้อมใช้งาน)

เครื่องจักรเดินเครื่องได้ตามเวลาที่กำหนดหรือไม่

บ่งชี้ว่าเครื่องจักรของคุณพร้อมใช้งานเมื่อควรจะทำงานหรือไม่

⚡

Performance (ประสิทธิภาพ)

ผลิตได้เร็วตามความเร็วมาตรฐานหรือไม่

แสดงว่าคุณกำลังผลิตในอัตราที่คาดหวังหรือไม่

✨

Quality (คุณภาพ)

ผลิตของดีได้มากน้อยแค่ไหน

บ่งชี้เปอร์เซ็นต์ของผลผลิตที่ยอมรับได้

กราฟ

แสดงแนวโน้ม OEE รายวัน/รายกะ และสาเหตุหลักที่ทำให้เครื่องหยุด (Top Downtime Reasons)

Manually Adding Downtime Records

Normally, downtime is automatically recorded when an operator stops production. However, you can also manually add downtime records.

📍 Navigation Path

การหยุดผลิต (Downtime)

1

Go to the การหยุดผลิต (Downtime) menu.

2

Click + เพิ่มบันทึก (+ Add Record).

3

Enter the following information:

  • • รหัสเครื่อง (Machine Code)
  • • เวลาที่เริ่มหยุด-สิ้นสุด (Start and End Time)
  • • สาเหตุ (Cause) - Examples: machine breakdown, waiting for materials, break time
4

Timeline View: The system will display a colored bar showing the time period when the machine was stopped, giving you a visual overview.

✅ Completion Checklist

  • ☑ Click Add Record button
  • ☑ Enter machine code
  • ☑ Set start and end time
  • ☑ Select/enter cause
  • ☑ View timeline visualization

Tips for Effective Monitoring

  • • Check the OEE Dashboard regularly to maintain the 85% target standard
  • • Review Top Downtime Reasons to identify recurring issues
  • • Use the Timeline view to spot patterns in downtime occurrences
  • • Monitor WIP to ensure production flow is smooth and on schedule
