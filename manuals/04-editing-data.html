Editing Saved Data

Modify existing master data and work orders

You can freely edit data that has already been saved in Mana MES. This guide explains how to edit Machine Data, Process Data, Production Routes, and SKU data.

To edit any master data:

1️⃣
Click Edit Button

(ปุ่มแก้ไข)

2️⃣
Enter New Information
3️⃣
Click Update

(อัปเดตข้อมูล)

Editing Machine Data (การแก้ไข ข้อมูลเครื่องจักร)

📍 Navigation Path

ข้อมูลอุปกรณ์ (Equipment Data)

1

Locate the machine you want to edit in the ข้อมูลอุปกรณ์ (Equipment Data) menu.

2

Click the Edit button (ปุ่มแก้ไข) for that machine.

3

Modify the machine details (process name and other information).

4

Click Update to save your changes.

  • ☑ Click Update to save

Editing Process Data (การแก้ไข ข้อมูลหลักกระบวนการผลิต)

📍 Navigation Path

ข้อมูลการผลิต > ข้อมูลหลักกระบวนการผลิต

(Production Data > Process Data)

1

Go to ข้อมูลการผลิต (Production Data) > ข้อมูลหลักกระบวนการผลิต (Process Data).

2

Find the process you want to edit and click the Edit button (ปุ่มแก้ไข).

3

Modify the process name and other details.

4

Click Update to save your changes.

✅ Completion Checklist
  • ☑ Navigate to Process Data
  • ☑ Locate process to edit
  • ☑ Click Edit button
  • ☑ Modify process details
  • ☑ Click Update to save

Editing Production Routes (การแก้ไข เส้นทางการผลิต)

📍 Navigation Path

ข้อมูลการผลิต > เส้นทางการผลิต

(Production Data > Production Routes)

1

Go to ข้อมูลการผลิต (Production Data) > เส้นทางการผลิต (Production Routes).

2

Find the production route you want to edit and click the Edit button (ปุ่มแก้ไข).

3

Modify the sequence of steps (Step) as needed.

4

Click Update to save your changes.

⚠️ Important Note

When editing Production Routes:

  • • Ensure step sequence is logical
  • • Keep steps complete
  • • Verify all steps are connected
✅ Completion Checklist
  • ☑ Navigate to Production Routes
  • ☑ Locate route to edit
  • ☑ Click Edit button
  • ☑ Modify step sequence
  • ☑ Verify sequence logic
  • ☑ Click Update to save

Editing SKU Data (การแก้ไข ข้อมูลหลักสินค้า (SKU))

📍 Navigation Path

ข้อมูลการผลิต > ข้อมูลหลักสินค้า (SKU)

(Production Data > SKU Master Data)

1

Go to ข้อมูลการผลิต (Production Data) > ข้อมูลหลักสินค้า (SKU) (SKU Master Data).

2

Find the SKU you want to edit and click the Edit button (ปุ่มแก้ไข).

3

Modify the product details (Product Code, Product Name, Unit of Measure) and components.

4

Click Update to save your changes.

📋 Editable SKU Fields
• Product Code
• Product Name
• Unit of Measure
• Components (Add/Remove)
✅ Completion Checklist
  • ☑ Navigate to SKU Master Data
  • ☑ Locate SKU to edit
  • ☑ Click Edit button
  • ☑ Modify product details
  • ☑ Update components if needed
  • ☑ Click Update to save

Tips for Editing Data

Bulk Updates

If you need to update multiple items, consider using the Bulk Upload feature with your previously used CSV file. The system will automatically detect and update the data.

Data Integrity

When editing Production Routes, ensure the sequence of steps remains logical and complete.

SKU Components

When editing SKU data, you can add or remove components as needed.

Work Orders

Edit work orders promptly if errors are detected to avoid production issues.
